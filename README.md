# OpenAI: LangChain

## ⚙️ Configuração do Ambiente

### Criando e Ativando o Ambiente Virtual

**Windows:**
```bash
python -m venv .venv
.venv\Scripts\activate
```

**Mac/Linux:**
```bash
python3 -m venv langchain
source .venv/bin/activate
```

### Instalação das Bibliotecas

```bash
pip install -r requirements.txt
```
